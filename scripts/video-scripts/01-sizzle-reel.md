# ECA Triage Accelerator - Sizzle Reel Script

**Duration**: 60 seconds
**Audience**: Marketing leads, LinkedIn, cold outreach
**Tone**: Confident, precise, no fluff. Legal professionals don't do hype.
**Resolution**: 1920x1080 @ 2x (Retina)
**Music**: Subtle, corporate-minimal (no EDM, no tension buildups)

---

## Pre-Roll (0-3s)

**Screen**: Black with ECA logo fade-in (Scale icon + "ECA Triage Accelerator")
**Narration**: None
**Transition**: Fade to app

---

## Scene 1: The Problem (3-8s)

**Screen**: Static graphic - "100 GB of documents. 72 hours to respond."
**Narration**: "Your opposing counsel just produced a hundred gigabytes. You have three days."
**Transition**: Hard cut

---

## Scene 2: Login + Matter Dashboard (8-14s)

**Screen**: App login -> matters list showing active cases with doc counts and status badges
**Click sequence**: Already logged in, show matters page with 3-4 matters
**Narration**: "ECA Triage Accelerator turns that mountain into the five gigs you actually need to review."
**Transition**: Crossfade

---

## Scene 3: Upload + Processing (14-22s)

**Screen**: Documents page with files uploading (progress bars, format icons)
**Click sequence**: Show upload zone -> files appearing in registry with status chips
**Narration**: "Ingest everything - PSTs, PDFs, Office docs, audio, video. Fifty-plus formats, no manual conversion."
**Highlight**: Format variety (show different file type icons in the registry)
**Transition**: Crossfade

---

## Scene 4: AI Brief (22-30s)

**Screen**: Matter brief page showing risk assessment, cost estimates, collection profile
**Click sequence**: Scroll through brief sections
**Narration**: "AI generates a case brief in minutes - risk assessment, cost estimates, processing plan. Download as branded PDF for your client."
**Highlight**: The PDF download button and branded output
**Transition**: Crossfade

---

## Scene 5: Hybrid Search (30-40s)

**Screen**: Search page with query "California energy crisis" -> results with relevance scores
**Click sequence**: Type query -> results populate -> hover on result to show X-ray preview
**Narration**: "Five search modes from keyword to full AI-powered RAG fusion. Hybrid search combines the precision of exact matching with the intelligence of semantic understanding."
**Highlight**: Mode selector showing all 5 modes, relevance scores on results
**Transition**: Crossfade

---

## Scene 6: Defensibility (40-48s)

**Screen**: Quick montage: citation card on a triage decision -> audit trail entry -> export dialog
**Click sequence**: Show citation card with source text highlighted, then cut to audit page
**Narration**: "Every AI decision is backed by citations. Full audit trail. Court-ready defensibility."
**Highlight**: Citation card showing exact source text
**Transition**: Crossfade

---

## Scene 7: Export (48-53s)

**Screen**: Export page showing Relativity/Everlaw/DISCO format options
**Click sequence**: Show export creation dialog with format selector
**Narration**: "Export production-ready load files for Relativity, Everlaw, or DISCO."
**Transition**: Crossfade

---

## Scene 8: Volume Reduction (53-57s)

**Screen**: Animated stat card: "100 GB -> 5 GB" with percentage reduction
**Narration**: "Ninety-five percent volume reduction. Before your first review platform bill."
**Transition**: Crossfade to closing

---

## Closing (57-60s)

**Screen**: ECA logo + "ecasses.com" + "Start Free Trial"
**Narration**: "ECA Triage Accelerator. Discovery at zero latency."
**Transition**: Fade to black

---

## Production Notes

- All screens use dark mode (the app default)
- Demo data: Use Enron matter on production server (3,000+ processed docs)
- No synthetic data - all search results and stats must be from real processed documents
- Record at 60fps, export at 30fps for smooth scrolling
- Cursor: Use a clean cursor theme, avoid system default
- Browser: Chrome in full-screen app mode (hide URL bar)
